# Django-Vue
前后端分离的练手项目

## 已实现功能：
1. 登录功能
2. 记账

## 待实现：
1. 博客
2. 读书
3. 其他

## 使用：
克隆
```
git clone https://github.com/WuAlin0327/Django-Vue.git
```
前端
```
cd frontend
npm install
```
打包文件
```
npm run build
```
后端
```
# 数据库迁移
python manage.py makemigrations
python manage.py migrate

# 启动后端
python manage.py runserver 
```

访问：
```
http://127.0.0.1:8000
```
新增用户：请手动添加
